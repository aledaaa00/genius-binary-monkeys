## Project Description
Disaster Eye is a centralized disaster monitoring platform that combines multiple natural hazards, including earthquakes, wildfires, floods, storms, and extreme heat, into a single solution. By integrating real-time data, AI-assisted analysis, safety guidance, and preparedness tools, Disaster Eye helps communities stay informed, reduce risks, and respond more effectively during emergencies.

## Problem Addressed
During natural disasters, critical information is often fragmented across multiple platforms, and communication networks can fail. Communities lack a unified, real-time command center that not only tracks global threats but also provides immediate, localized offline survival logistics, automated medical triage, and interactive safety education for children in one centralized interface.

## Features
Global Threat Radar with real-time NEXRAD weather overlays and interactive Leaflet mapping
S.O.S Emergency Broadcasting and Volunteer Rescue Network registration system
Bio-Triage AI for instant medical first-aid protocols based on trauma input
Real-Time Seismic Equalizer tracking USGS data with localized proximity threat calculations
Tactical Survival Logistics Calculator for grid-down inventory management
Hydro-Sensors and Wildfire Tracking for simulated environmental spread analysis
Cyber Scout Academy offering child-friendly interactive disaster preparedness training

## Technologies Used
Python, Flask, HTML5, Vanilla JavaScript, CSS3, Bootstrap 5, Leaflet.js, CartoDB Maps, USGS GeoJSON APIs, IEM NEXRAD WMS Radar, Nominatim OpenStreetMap API.

## How to run the Project
Ensure Python 3 is installed on your computer. Open a terminal or command prompt in the main project folder. Type "pip install flask requests" and press Enter to install the required dependencies. Type "python app.py" and press Enter to start the local server. Open any web browser and access http://localhost:5001 to view the dashboard.
